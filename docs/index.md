# AutoJoin for SteamGifts

🎮 **Extensão automática para participar de sorteios no SteamGifts**

## 📋 Sobre

O AutoJoin for SteamGifts é uma extensão de navegador que automatiza a participação em sorteios no site SteamGifts.com, permitindo que você participe de múltiplos sorteios de forma eficiente e organizada.

## ✨ Funcionalidades Principais

- 🤖 **Auto-join automático** em sorteios
- 🎯 **Filtros inteligentes** por pontos, nível e wishlist
- 🎨 **Interface personalizável** com tema escuro
- ⚙️ **Configurações avançadas** de automação
- 🔒 **Operação segura** respeitando limites do site

## 📦 Instalação

1. Baixe a extensão do [GitHub Releases](https://github.com/bernardopg/AutoJoin-for-SteamGifts/releases)
2. Extraia o arquivo ZIP
3. Abra o Chrome/Firefox e acesse as extensões
4. Ative o "Modo do desenvolvedor"
5. Clique em "Carregar extensão descompactada"
6. Selecione a pasta extraída

## 🚀 Como Usar

1. Instale a extensão
2. Acesse [SteamGifts.com](https://www.steamgifts.com)
3. Configure suas preferências nas configurações da extensão
4. A extensão começará a funcionar automaticamente

## 🔧 Configurações

A extensão oferece diversas opções de configuração:

- **Filtros de Pontos**: Define quantidade mínima/máxima de pontos
- **Filtros de Nível**: Participa apenas de sorteios do seu nível
- **Wishlist Only**: Foca apenas em jogos da sua wishlist
- **Tema**: Interface clara ou escura
- **Velocidade**: Controla velocidade de automação
